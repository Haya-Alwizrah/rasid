# نظام راصد

الوصف
نظام راصد هو نظام ذكي يكشف حالات الاحتيال في التأمين الصحي باستخدام الذكاء الاصطناعي، من خلال تحليل بيانات المريض وربطها بين الاستقبال، العلامات الحيوية، والطبيب، ثم مقارنتها مع بياناته الأساسية وزياراته السابقة لاكتشاف أي تناقضات.

التشغيل:
* إدخال بيانات المريض من قبل أقسام المستشفى
* تحليل البيانات تلقائيًا داخل النظام
* مقارنة البيانات الحالية مع السجلات السابقة
* عرض نتيجة توضح ما إذا كانت الحالة طبيعية أو تحتوي على شبهة احتيال
