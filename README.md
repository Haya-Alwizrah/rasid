# نظام راصد

الوصف
نظام راصد هو نظام ذكي يكشف حالات الاحتيال في التأمين الصحي باستخدام الذكاء الاصطناعي، حيث يقوم النظام بتحليل المطالبات الطبية والتقارير المرتبطة بها لاكتشاف اي تناقضات أو انماط غير طبيعيه قد تشير إلى الشخص الذي يستخدم التأمين ليس صاحبه الحقيقي. من خلال تحليل بيانات المريض وربطها بين الاستقبال، العلامات الحيوية، والطبيب، ثم مقارنتها مع بياناته الأساسية وزياراته السابقة لاكتشاف أي تناقضات.


 يعتمد النظام على كل من النموذج اللغوي الكبير "نهى" المقدم من شركة علم بالإضافة إلى تقنية استرجاع المعلومات (RAG).

التشغيل:
* إدخال بيانات المريض من قبل أقسام المستشفى
* تحليل البيانات تلقائيًا داخل النظام
* مقارنة البيانات الحالية مع السجلات السابقة
* عرض نتيجة توضح ما إذا كانت الحالة طبيعية أو تحتوي على شبهة احتيال
